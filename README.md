# FoodTrack - Base de Datos

Plataforma para gestionar operaciones de **foodtrucks** en una ciudad.

## Descripción del Modelo Relacional

### Entidades principales:
- **Foodtrucks**: Información de los camiones de comida.
- **Products**: Productos ofrecidos por cada foodtruck.
- **Locations**: Ubicaciones históricas de los foodtrucks.
- **Orders**: Pedidos realizados.
- **Order_Items**: Detalle de los ítems de cada pedido.

### Tecnologías
- Microsoft SQL Server
- DBeaver (cliente)
- Git + GitHub

## Estructura del proyecto
- `/scripts/` → Scripts SQL de creación y evolución del esquema
- `/data/` → Archivos CSV fuente
